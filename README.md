# DefensiveTokens
Defending Against Prompt Injection With a Few DefensiveTokens предлагается тестовая (test-time) защита от prompt injection, основанная на добавлении небольшого числа специально обученных DefensiveTokens — новых токенов с оптимизированными эмбеддингами, которые препендятся ко входу LLM.
